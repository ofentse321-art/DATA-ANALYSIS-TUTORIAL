1. What is Data Analysis? 🤔

Data means information. Analysis means studying or looking at something carefully.

So, Data Analysis means: looking at information to understand it better and make good decisions.

2. Why is Data Important? 💡

We use data every day!

  • When we count how many people like a song 🎵
  • When we check our school marks 📚
  • When a shop checks which toy sells the most 🧸

3. How Do We Analyze Data? 🔍

We do these steps:

  1. Collect data (Get information)
  2. Organize it (Put it in a table or list)
  3. Study it (Find patterns or answers)
  4. Show results (Draw graphs or write a summary)

4. What is a Dataset? 📊

A dataset is just a group of data — like a table full of numbers or facts.

For example, we can make a dataset about favorite fruits of 5 students:

Name Favorite Fruit
Anna Apple
Ben Banana
Clara Apple
David Grapes
Ella Banana

5. Let's See an Example! 🧮

Let’s count how many people chose each fruit:

  • 🍎 Apple = 2
  • 🍌 Banana = 2
  • 🍇 Grapes = 1

6. Step-by-Step Analysis 🪜

  1. We looked at each person’s favorite fruit.
  2. We counted how many liked Apple, Banana, and Grapes.
  3. We found that Apple and Banana were the most liked.
  4. Now we can tell a shop: “Buy more apples and bananas!”
